## Brief Description
This project contains basic HTML exercises created for a web development assignment.
It demonstrates the use of HTML elements such as headings, paragraphs, lists, images,
forms, tables, and semantic HTML5 elements.

## Steps to Run the Project
1. Download or clone the repository from GitHub.
2. Open the project folder.
3. Open any `.html` file (exercise1.html, exercise2.html, etc.) in a web browser.
4. Alternatively, open the files using VS Code and run them with Live Server.
